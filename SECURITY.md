# Security Policy

## What this repository is
This repository contains a public JD-to-scorecard skill and the HRClaw product docs around it.

## What must not be added
Do not commit:
- API keys
- browser session data
- login cookies
- private candidate data
- internal ATS or CRM exports
- client-specific scoring rules

## Reporting a security issue
Please report vulnerabilities privately through GitHub Security Advisories or a direct maintainer contact.

Include:
- a short summary
- affected file(s)
- reproduction steps
- impact
- suggested fix

## Safe usage guidance
- Treat third-party skills as untrusted content.
- Review prompts before using them on sensitive data.
- Keep public examples anonymous or synthetic.
- Separate the public skill from any private hiring workflow.
