# Changelog

All notable changes to this project will be documented in this file.

## [0.2.0] - 2026-04-08

### Added
- Recruiter-facing trial hub flow as the default post-login landing page.
- Apple-inspired admin UI refresh across login, navigation, tasks, and high-frequency screening pages.
- Local-first product positioning for the full HRClaw workflow in the root README files.
- Pilot execution checklist and trial SOP for HR validation.
- Browser capture plugin positioned as part of the same HRClaw screening workflow.

### Changed
- Fixed bulk resume import file selection by replacing the hidden-input JavaScript click flow with a native label-triggered file picker.
- Refreshed packaged frontend and Chrome extension artifacts so install bundles match the current UI.
- Updated project metadata to reflect the broader HRClaw product instead of the original week-one MVP naming.

## [0.1.0] - 2026-03-28

### Added
- Public `jd-scorecard` skill package.
- JD to scorecard prompt and interview-question prompt.
- Structured scorecard template and example outputs.
- MIT license, security policy, and skill validation workflow.
- GitHub homepage logo and public-facing open-source positioning.
