# 项目概要：HRClaw Remotion Promo

## 一、项目概述
HRClaw Remotion Promo 是用于生成 HRClaw 产品宣传短片的 Remotion 项目（React + Remotion）。视频以真实产品界面为主，配合简洁文案与旁白，面向招聘负责人与HR场景的展示。

## 二、输出规格
- Composition ID：`HRClawPromo1080p`
- 分辨率：1920 × 1080
- 帧率：30 fps
- 预期时长：约 50 秒
- 渲染输出（脚本）：`renders/hrclaw-promo.mp4`（见 `package.json` 中的 `render` 脚本）

## 三、目标与故事流程
1. 品牌开场（Brand intro）
2. 登录与试点中心（Login / Trial hub）
3. `jd-scorecard` Skill 演示（把 JD 转成评分卡）
4. Scorecard 管理（统一筛选口径）
5. 批量简历导入并打分（Batch import & scoring）
6. 批次结果概览（命中项 / 缺失项 / 建议）
7. 浏览器侧边栏捕获（侧边栏采集候选人页面）
8. CTA 结束（Outro）

目的：展示从 JD、简历到浏览器页面能进入同一评分工作流，突出节省 HR 时间并保证复核证据。

## 六、运行与开发

```bash

```

```bash

```

```bash
npm run render
```

- 类型检查与 lint：

```bash
npm run lint
```

## 七、编辑要点与注意事项
- 场景时长由 `SCENES` 常量控制，合成总时长由 `TOTAL_DURATION` 计算得出，修改 SCENES 会影响时间轴。
- `ClipScene` 使用 `OffthreadVideo` 播放素材并在面板上展示标题、要点、进度条与 redaction mosaic，调整面板样式在 `panelBase` 与 `pillStyle`。
- 若需隐藏真实候选人隐私信息，可在 `ClipScene` 的 `redactions` 中配置位置与大小。
- 品牌色与阴影变量集中在 `theme.ts`，便于统一风格调整。
- `remotion.config.ts` 已启用 Tailwind（`@remotion/tailwind-v4`），如需使用 Tailwind 类名请检查 `index.css` 与 Tailwind 配置。
